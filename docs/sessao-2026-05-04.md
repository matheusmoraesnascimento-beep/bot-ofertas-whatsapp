## Status geral

## O que foi implementado

### Plano 1 — Foundation
- `db.py` — SQLite substituindo CSV. Tabelas: `ofertas_enviadas`, `historico_precos`
- `filtros.py` — migração automática do CSV para SQLite na primeira execução
- `whatsapp.py` — retry 3x com 4 seletores fallback para envio com imagem

### Plano 2 — Painel + Deploy
- `painel/app.py` — Flask dashboard com auth, rotas REST
- `painel/auth.py` — decorator `@login_required`, senha via `PAINEL_SENHA`
- `painel/state.py` — `bot_state.json` para status, pausa, forçar rodada
- `painel/templates/login.html` e `index.html` — dashboard single-page
- `Dockerfile` — python:3.12-slim + Playwright + Chromium
- `railway.toml` — builder dockerfile, restart on failure
- `start.py` — sobe bot.py + painel em paralelo

### Plano 3 — Expansão
- `inteligencia.py` — score composto + agenda por categoria + histórico de preço mínimo
- `mercadolivre.py` — OAuth com auto-refresh de token
- `kabum.py` — scraping via ScraperAPI (fix: URL não deve ser double-encoded)
- `bot.py` — integrado com tudo acima
